# Atualização: fase de grupos completa da Copa 2026

Este patch completa a base do repositório com os 8 jogos que faltavam: Egito 1 x 1 Irã, Nova Zelândia 1 x 5 Bélgica, Panamá 0 x 2 Inglaterra, Croácia 2 x 1 Gana, Argélia 3 x 3 Áustria, Jordânia 1 x 3 Argentina, Colômbia 0 x 0 Portugal, RD Congo 3 x 1 Uzbequistão.

Validação: `data/resultados.csv` e `data/resultados_reais.csv` agora têm 72 jogos finalizados; `data/desempenho/resumo_desempenho_por_jogo_copa_2026.csv` também tem 72 linhas; `data/desempenho/jogadores_citados_desempenho_copa_2026.csv` tem 299 registros.

Não foram criadas notas/rating quando as fontes não traziam esse dado padronizado. Foram registrados apenas eventos verificáveis: gols, assistências quando relatadas, cartões quando relatados, defesas, pênaltis, gols anulados, lesões e destaques citados em match reports.
