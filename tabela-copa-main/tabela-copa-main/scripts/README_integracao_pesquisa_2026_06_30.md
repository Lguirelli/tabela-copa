# Integração manual da pesquisa Copa 2026

Este repositório já contém os CSVs normalizados gerados a partir da pesquisa `docs/Pesquisa aprofundada da Copa do Mundo FIFA 2026.pdf`.

Arquivos principais adicionados:
- data/desempenho_jogo_a_jogo.csv
- data/desempenho_fase_grupos_selecoes.csv
- data/desempenho_jogadores_destaques.csv
- data/desempenho_penaltis.csv
- data/fontes_dados.csv
- data/pendencias_dados.csv
- data/auditoria_dados.csv

Observação: o jogo_id de mata-mata foi normalizado ao repositório:
- R32_75 = Países Baixos x Marrocos
- R32_76 = Brasil x Japão
